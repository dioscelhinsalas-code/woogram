# Memorama de Profesiones y Oficios 👩‍🚀🧑‍🚒

Juego educativo de memoria (memorama) para niños, enfocado en el reconocimiento de **profesiones y oficios**. Incluye audio interactivo, tres niveles de dificultad y animaciones alegres.

## 🎮 Jugar en línea

**[▶ Jugar ahora](https://dioscelhinsalas-code.github.io/woogram/memorama-profesiones/)**

## ✨ Características

| Función | Descripción |
|---|---|
| 🔊 **Audio con nombre** | Cada vez que encuentras una pareja, el juego pronuncia el nombre de la profesión en español |
| 🎵 **Efectos de sonido** | Pop al voltear, fanfarria al acertar, sonido tierno al fallar (Web Audio API) |
| 🎭 **3 Niveles** | Fácil (3 parejas), Medio (6 parejas), Difícil (10 parejas) |
| 🔀 **Reorganización automática** | Las tarjetas se barajan al cambiar de nivel |
| 🎉 **Modal de felicitación** | Al terminar cada nivel aparece un mensaje con opción de avanzar al siguiente |
| 🌈 **Diseño infantil** | Colores vivos, animaciones de brinco y confeti animado |
| 📱 **Responsivo** | Funciona en móviles, tablets y escritorio |

## 🚀 Ejecutar localmente

Solo abre `index.html` en cualquier navegador moderno. No requiere servidor ni dependencias.

```bash
# Con Python (opcional)
python3 -m http.server 8080
# Luego visita http://localhost:8080
```
